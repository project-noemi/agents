# Marketing Agents (Documentation)

## Overview
This directory contains documentation for agents specialized in brand strategy, SEO, and content creation.

## Personas
- **Brand Strategist**: Develops and maintains the "Project NoéMI" brand identity.
  - Spec: `agents/marketing/brand-strategist.md`
- **SEO Strategist**: Optimizes content for search engines.
  - Spec: `agents/marketing/seo-strategist.md`
- **Video Content Manager**: Manages video workflows and automation.
  - Spec: `agents/marketing/video-content-manager.md`
- **Thumbnail Specialist**: Creates high-conversion video thumbnails.
  - Spec: `agents/marketing/thumbnail-specialist.md`
